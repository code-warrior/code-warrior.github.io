Sublime Text 3 User Preferences

The following files must go in your User folder, which is inside the Packages folder.

  • C/C++ Sublime settings
  • CSS Sublime settings
  • JSLint Sublime settings
  • Java Sublime settings
  • JavaScript Sublime settings
  • PHP Sublime settings
  • Plain text Sublime settings
  • Preferences Sublime settings
  • UNIX shell Sublime settings
